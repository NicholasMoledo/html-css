Estilos CSS internos / locais

- A tag <style> é criada dentro da tag <head>

- Um parâmetro tag{ } para cada <tag> a ser formatada

 Ex:	<head>
		<style>
        		body {
            		background-color: lightsteelblue;
            		font-family: Arial, Helvetica, sans-serif;
            		font-size: 15px;
        		}

        		h1 {
            		color: mediumblue;
            		background-color: darksalmon;
        		}

        		h2 {
            		color: darkred;
        		}

        		p {
            		text-align: justify;
        		}
    		</style>
	</head>
